The Combobox is inspired by [GitHub](https://github.com)'s PR label selector and is powered by [shadcn/ui](https://ui.shadcn.com). Almost all elements are [radix-ui](https://www.radix-ui.com) components, styled with [tailwindcss](http://tailwindcss.com/).

It uses the following components:

- [Accordion](https://ui.shadcn.com/docs/components/accordion)
- [Badge](https://ui.shadcn.com/docs/components/badge)
- [Button](https://ui.shadcn.com/docs/components/button)
- [Command](https://ui.shadcn.com/docs/components/command) _(powered by [cmdk](https://cmdk.paco.me))_
- [Dialog](https://ui.shadcn.com/docs/components/dialog)
- [Input](https://ui.shadcn.com/docs/components/input)
- [Label](https://ui.shadcn.com/docs/components/label)
- [Popover](https://ui.shadcn.com/docs/components/popover)

The **only** change to the core files was adding a smaller `Button` size:

```js
// Button.tsx
const buttonVariants = cva(
  variants: {
    size: {
      xs: "h-8 px-2 rounded-md", // added
      // ...
    }
    // ..
  }
  // ..
)
```

The source code is available on [GitHub](https://github.com/mxkaske/mxkaske.dev/tree/main/components/craft/fancy-box.tsx).
